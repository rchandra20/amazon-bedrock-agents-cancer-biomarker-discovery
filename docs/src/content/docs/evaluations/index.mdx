Agent evaluation allow you to run evaluation jobs with multiple agents that evaluate both agent goals and task accuracy

## Available datasets

Trajectories for the cancer biomarker discovery workflow 

## Available metrics

- **[Agent goal]** Chain of thought evaluation of agent rationale and predeifned instructions

- **[Text2SQL Task accuracy]** Text2SQL metrics for sql equivalence and factual correctness

- **[RAG Task accuracy]** RAG accuracy for contextual accuracy and factual correctness 

- **[Custom Task accuracy]** Bring your own metric for a specific downstream task of an agent
